# Documentación del Proyecto

## 1. Documentación del Backend

### Estructura de Carpetas
- **backend/**: Contiene el código del servidor.
  - **src/**: Código fuente del backend.
    - **app.js**: Archivo principal de la aplicación.
    - **config/**: Configuraciones de la base de datos y otros servicios.
    - **controllers/**: Controladores que manejan la lógica de negocio.
    - **models/**: Modelos de datos que representan las entidades.
    - **routes/**: Rutas de la API.
    - **middleware/**: Middleware para la autenticación y otras funciones.
    - **utils/**: Utilidades y funciones auxiliares.
    - **validators/**: Validadores para las entradas de datos.

### Instrucciones de Configuración
1. Clonar el repositorio.
2. Instalar las dependencias con `npm install`.
3. Configurar las variables de entorno necesarias.
4. Ejecutar el servidor con `node src/app.js`.

## 2. Documentación del Frontend

### Estructura de Carpetas
- **frontend/**: Contiene el código del cliente.
  - **src/**: Código fuente del frontend.
    - **pages/**: Páginas de la aplicación.
    - **components/**: Componentes reutilizables.
    - **services/**: Servicios para la comunicación con la API.

### Instrucciones de Configuración
1. Clonar el repositorio.
2. Instalar las dependencias con `npm install`.
3. Ejecutar la aplicación con `npm start`.
